Node.js und Express

Die Ordner in diesem Verzeichnis enthalten alle eine Datei package.json, in der die Abhängigkeiten (d.h. die benötigten Module) für die jeweiligen Node.js Anwendungen vermerkt sind.

Wechseln Sie auf der Kommandozeile in die jeweiligen Ordner und installieren Sie die Packages mit dem folgenden Kommando:

npm install
